# No future Scala 2.10.x releases are planned.
[Scala 2.10.6](https://github.com/scala/scala/releases/v2.10.6) concluded this series.

We encourage you to target 2.12.x or 2.13.x instead. If you're feeling nostalgic, check out the [the 2.10.x contribution guidelines](https://github.com/scala/scala/blob/v2.10.5/CONTRIBUTING.md)!
